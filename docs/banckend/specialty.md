<a name="create"></a>
#### 添加专业
<pre>
POST /specialty
</pre>
##### 请求参数

| 属性名称 | 必填 | 说明 | 备注 |
|:---:|:---:|:---:|:---:|
| name | 是 | 专业名称 | - |
| course_ids | 是 | 课程id | 多个课程用逗号拼接 |
