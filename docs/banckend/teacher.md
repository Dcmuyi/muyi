<a name="search"></a>
#### 教师列表
<pre>
POST /teacher/search
</pre>
##### 请求参数

| 属性名称 | 必填 | 说明 | 备注 |
|:---:|:---:|:---:|:---:|
| keyword | 否 | 关键词 | 手机、姓名都可以搜索 |

-------------------------
<a name="create"></a>
#### 添加教师
<pre>
POST /teacher/create
</pre>
##### 请求参数

| 属性名称 | 必填 | 说明 | 备注 |
|:---:|:---:|:---:|:---:|
| mobile | 是 | 手机号 | - |
| type | 是  | 类型 | 20：教师 |
| name | 是  | 姓名 | - |
